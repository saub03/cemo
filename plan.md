넌 이제부터 Python TUI(Terminal User Interface) 및 시스템 유틸리티 개발 전문가야. 
나는 'Command-Flow'라는 터미널 커맨드 치트시트 프로그램을 개발하려고 해. 
아래의 기획서와 요구사항을 꼼꼼히 읽고, 단계별로 완벽하게 동작하는 파이썬 코드를 작성해줘.

# 1. 프로젝트 개요 (Command-Flow)
- **목표:** 개발자가 자주 쓰는 명령어를 마우스 없이 빠르게 검색하고, 동적 변수를 입력받아 클립보드에 자동 복사하는 키보드 중심의 터미널 앱.
- **핵심 특징:** Textual을 이용한 미려한 TUI, 동적 변수 치환(`{{변수}}`), 전역 단축키를 통한 빠른 호출.

# 2. 기술 스택 및 라이브러리
다음 라이브러리들을 반드시 사용하여 구현해야 해.
- **UI 프레임워크:** `textual` (마우스 없는 100% 키보드 조작 지원, 화면 분할, 모달 창 구현)
- **데이터베이스:** `sqlite3` (내장 DB, 명령어 저장 및 검색)
- **전역 단축키:** `keyboard` (백그라운드에서 단축키 감지하여 창 활성화)
- **클립보드 제어:** `pyperclip` (완성된 명령어 자동 복사)
- **검색 알고리즘:** `thefuzz` (퍼지 검색으로 오타나 초성 검색 지원)

# 3. 데이터베이스 스키마 설계 (SQLite)
`commands` 테이블은 다음과 같은 구조를 가져야 해.
- `id` (INTEGER, PK)
- `large_category` (TEXT) - 대분류 (예: git)
- `medium_category` (TEXT) - 중분류
- `small_category` (TEXT) - 소분류
- `command` (TEXT) - 명령어 (예: `git commit -m "{{message}}"`)
- `description` (TEXT) - 설명
- `tags` (TEXT) - 태그 (쉼표로 구분)
- `usage_count` (INTEGER) - 사용 횟수 (자동 정렬용)

# 4. 핵심 기능 요구사항
1. **화면 레이아웃 (Textual):**
   - 왼쪽 사이드바: 대분류/즐겨찾기 트리 뷰
   - 메인 화면: 선택된 분류의 명령어 목록 (DataTable)
   - 헤더/푸터: 현재 상태 및 단축키 안내 (`/`: 검색, `A`: 추가 등)
2. **동적 변수 치환 및 클립보드 복사 (핵심 로직):**
   - 사용자가 DataTable에서 특정 명령어를 선택(Enter)했을 때, 해당 `command` 문자열 안에 `{{ }}`로 감싸진 변수가 있는지 정규식으로 검사.
   - 변수가 있다면, Textual의 `ModalScreen`을 띄워 사용자에게 해당 변수 값을 입력받음.
   - 변수가 없거나 입력이 완료되면, 최종 완성된 문자열을 `pyperclip.copy()`로 복사하고, 하단에 "복사 완료" 스낵바(Notification)를 띄움.
3. **글로벌 단축키 (백그라운드):**
   - 프로그램 실행 시 백그라운드 스레드로 `keyboard.add_hotkey('alt+space', ...)`를 등록.
   - 단축키 입력 시 현재 터미널 창을 최상단으로 활성화 (OS에 맞는 포커스 로직 적용).

# 5. 개발 진행 방식 (중요)
한 번에 모든 코드를 작성하면 너무 길어지니, 아래 **단계별(Step-by-Step)로 하나씩 코드를 작성해줘.** 
내가 "다음 단계 진행해"라고 말할 때마다 다음 스텝의 코드를 줘.

- **Step 1:** `sqlite3`를 이용한 DB 초기화 및 테스트용 더미 데이터 생성 함수 작성 (DB 매니저 클래스).
- **Step 2:** `Textual`을 이용한 기본 UI 레이아웃 구성 (사이드바, DataTable 렌더링). Step 1의 DB와 연동하여 데이터를 화면에 뿌려줄 것.
- **Step 3:** Textual 키보드 이벤트 처리 및 **동적 변수 치환용 Modal 팝업창** 로직 구현 (`pyperclip` 복사 포함).
- **Step 4:** `thefuzz`를 이용한 검색창 기능 추가 및 `keyboard`를 이용한 글로벌 핫키 스레드 통합.
